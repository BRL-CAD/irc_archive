00:44.38Notifythat search would look like gdiff2 -F "-type tgc" file1.g file2.g and would report only items that were tgc primitives at some stage of the diffing process. I.e., an added tgc in file2.g would be reported, as would a primitive named s.1 that was changed from a tgc in file1.g to an ell in file2.g. This should provide a way to trim potentially large diff results down to what is of current interest to a user.
